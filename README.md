Aplikasi ini dibuat untuk menyelesaikan tugas So 
Hanya gambar lingkaran yang bergerak memantul 
tidak menggunakan keyframe tetapi menggunakan thread.

fitur yang akan ditambahkan:
-button create lingkaran
-button untuk run
-button interrupt
-button Kill
